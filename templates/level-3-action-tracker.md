## Purpose

**The single canonical list of every open action item you own.**

Before this file exists, actions tend to scatter — some in your priority tables, some
buried inside session-log prose, some only in your head. The result is a persistent,
low-grade feeling that you're forgetting something. Usually, you are.

This file is the **action extract**: one place, scannable in 30 seconds, surfaced at the
start of every session. Full context still lives in your main SOT priorities — this is
just the "what do I need to actually *do*" layer.

**Rules:**
- **Action layer only.** Items you own (or drive). Anything fully delegated to someone
  else lives in *their* entity file, not here.
- **One row per action:** status · action · due/trigger · source · cross-reference.
- **Status taxonomy:**
  - 🔴 overdue / urgent
  - 🟠 active this week
  - 🟡 open, no deadline
  - 🟢 done (keep one session, then archive)
  - ⏸️ deferred / waiting on something or someone
- **AI maintains it:** any newly-captured action is added the moment it appears in a
  session. Closed items get marked 🟢, then archived at the next session.
- **Surfaced every session start** (add this to your session ritual / router).
- **Does NOT replace** your main priority tiers — those carry full context, watchlist,
  and delegated items. This is the fast-scan extract.
